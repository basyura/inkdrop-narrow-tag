# switch-tag

Inkdrop's plugin to switch by tag.

![Screenshot](https://raw.githubusercontent.com/basyura/inkdrop-switch-tag/master/images/image.png)

## Install

```
ipm install switch-tag
```

## Keybindings

| Command         | Explanation       |
| --------------- | ------------------|
| switch-tag:open | open tag selector |
