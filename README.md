# narrow-tag

Inkdrop's plugin to narrow by tag.

![Screenshot](https://raw.githubusercontent.com/basyura/inkdrop-narrow-tag/master/images/image.png)

## Install

```
ipm install narrow-tag
```

## Keybindings

| Command         | Explanation       |
| --------------- | ------------------|
| narrow-tag:open | open tag selector |
